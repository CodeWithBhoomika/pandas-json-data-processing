# pandas-json-data-processing
### AIM: To load data from a JSON file using Pandas, convert it into a DataFrame, process the data, and save the processed data back into JSON format.

### TOOLS USED
* Python
* Pandas Library
* Google Colab

### ALGORITHM
1. Import the Pandas library.
2. Read the JSON file using `pd.read_json()`.
3. Convert JSON data into a DataFrame.
4. Perform data processing (example: add a new column).
5. Save the processed DataFrame back into JSON format using `to_json()`.

### FILES IN THIS REPOSITORY
* **Pandas_JSON_Experiment.ipynb** – Python notebook containing the code.
* **input_data.json** – Input JSON dataset.
* **processed_data.json** – Output JSON file generated after processing.

### RESULT
The JSON data was successfully loaded into a Pandas DataFrame, processed, and saved back into a new JSON file.
